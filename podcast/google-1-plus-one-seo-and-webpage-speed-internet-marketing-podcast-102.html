Questions this episode answers

What is Google's Plus One button and how does it work?

The Plus One button is Google's equivalent of the Facebook like button. It appears in Google SERPs, on ads, and on select websites. Users must be signed into a Google account to see and use it. Clicking it turns the button blue and adds the result to the Plus Ones tab on your Google profile.

Will Google Plus One affect search rankings?

Google had not explicitly confirmed that Plus One votes affect SERPs at the time of this episode. However, the hosts expect Plus One data tied to social connections will eventually influence results, similar to how Twitter retweets were already appearing in Google search results.

How can I check and improve my website page speed?

Go to pagespeed.googlelabs.com, enter your URL, and the tool will audit your page. It categorizes recommendations as high, medium, or low priority. High priority items include optimizing images and minimizing redirects, which offer the largest performance gains for the least development effort.

Is a table or a list better for SEO when displaying a price list?

According to the hosts, list items may be slightly more SEO friendly because search engines give special consideration to lists. However, for presenting a price chart clearly, using a table is more effective. They suggest using a table header row and experimenting with placing an H3 tag inside that header.

Terms defined in this episode
Plus One button
Google's social endorsement button, similar to the Facebook like button, shown in SERPs and ads for signed-in U.S. users, adding clicked results to a user's Google profile Plus Ones tab.
PageSpeed Labs
A Google beta tool at pagespeed.googlelabs.com that analyzes a webpage URL and returns prioritized recommendations, labeled high, medium, or low, for improving page load speed.
OpenSiteExplorer
A site analysis tool at opensiteexplorer.org that reports page authority, domain authority, linking root domains, and total links for any given URL.
CSS sprites
A technique mentioned in PageSpeed Labs output where multiple images are combined into a single image and referenced via CSS, reducing page load time.
