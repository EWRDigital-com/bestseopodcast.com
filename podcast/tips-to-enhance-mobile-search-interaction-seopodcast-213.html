Tips to Enhance Mobile Search Interaction - #seopodcast 213

Episode30 min2014-01-02

To improve mobile search, make phone numbers tappable, list core services prominently, use autocomplete and infinite scrolling, provide localized information, and always include social media links and a link to the full site.

Apple PodcastsSpotifyYouTubeiHeartRadioRSS
Key takeaways
  • All phone numbers on mobile sites should be linked so users can tap to call instantly.
  • List only core services prominently on mobile; hide secondary pages like FAQ and About Us.
  • Use infinite scrolling on mobile instead of pagination for a smoother one-handed experience.
  • Include social media links on mobile pages so users can check in, like, or review right from their device.
  • Always provide a link to the full site so mobile users who want more information can access it.

Questions this episode answers

What are the biggest mistakes to avoid when building a mobile site?

Common mistakes include failing to redirect links correctly when hosting a mobile version on a separate domain like m.yoursite, and not configuring canonical links properly, which can create error loops, prevent pages from loading on phones, and cause ranking losses according to the hosts.

Should you build a fully separate mobile site or use responsive design?

The hosts argue responsive design is most important for mobile phones, especially portrait orientation. Desktop versions work fine on tablets due to pinch-to-zoom, so spending design resources there is less valuable. The real investment should go into the mobile phone experience where conversions happen.

How should social media be incorporated into a mobile search experience?

The hosts say social should be included on every mobile page. For example, a location page can prompt users to check in on Foursquare for a discount, like the business on Facebook, or leave a review on Google Plus, with all relevant social links placed directly on that page.

Terms defined in this episode
Image Mismatch
A Google Webmaster Tools penalty issued when the image displayed on a website differs from what Google indexes for image search, which can result in images being removed from search results.
Infinite Scrolling
A mobile design approach where new content or images automatically load at the bottom of the page as the user scrolls, removing the need to tap through paginated pages.
Responsive Design
A web design approach that adapts a site's layout to different screen sizes; the hosts recommend prioritizing it for mobile phone portrait orientation rather than for tablets or desktops.
Canonical Links
Configuration that tells search engines which version of a URL is the primary one, important when running a separate mobile domain to avoid error loops and ranking issues.
