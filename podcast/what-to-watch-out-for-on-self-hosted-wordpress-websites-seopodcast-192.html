What to watch out for on Self Hosted Wordpress Websites - #seopodcast 192

Episode31 min2013-07-03

Self-hosted WordPress SEO requires using a dedicated plugin like Yoast, proper permalink structures, taxonomy descriptions, image alt text, authorship claims, and custom meta descriptions to avoid common on-page mistakes.

Apple PodcastsSpotifyYouTubeiHeartRadioRSS
Key takeaways
  • Use Yoast WordPress SEO plugin instead of relying on theme-built SEO tools, which lose all settings if you change themes.
  • Set up a proper permalink structure using post names, and customize slugs to include your targeted keyphrase.
  • Fill in image title, caption, alt text, and description fields every time you upload an image to WordPress.
  • Claim Google Plus authorship for your content because it increases click-throughs from search results.
  • Change your WordPress username from admin and use a strong alphanumeric password to block brute-force attacks.

Questions this episode answers

Which WordPress SEO plugin should I use for a self-hosted site?

The hosts recommend Yoast, also called WordPress SEO by Yoast. They previously used All-in-One SEO and SEO Ultimate, but settled on Yoast because it has all the features of both and keeps everything visible and easy to change, with authorship automation built in.

How do I protect a self-hosted WordPress site from brute-force attacks?

Change your username from the default admin to something else, use a strong alphanumeric password with symbols, and install a plugin that tracks and locks out IP addresses after multiple failed login attempts. The plugin can lock out IPs for five minutes, then 20 minutes, then 24 hours.

What is the difference between WordPress posts and pages for SEO?

Pages are meant for static content like About Us or Contact. Posts are for content you want shared, syndicated via RSS, or sent in newsletters. Using pages for blog content breaks RSS feeds and sharing, so time-sensitive or shareable content should always be published as posts.

Terms defined in this episode
Taxonomy
A way to group similar content in WordPress, such as categories and tags, whose description fields can rank in search and appear as SERP descriptions.
Permalink
The permanent URL structure for a WordPress post or page. Setting it to post name instead of a query string makes URLs readable and search engine friendly.
Slug
The customizable portion of a WordPress permalink that lets you shorten or target a specific keyphrase in the URL without changing the post title.
Authorship
A Google Plus feature that connects content to a creator's profile, pulling their avatar into search results and increasing click-through rates.
