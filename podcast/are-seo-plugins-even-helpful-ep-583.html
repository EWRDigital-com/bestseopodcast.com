Are SEO Plugins even Helpful? Ep. 583

Ep. 58321 min2023-10-152,255 plays

SEO plugins like Yoast and Rank Math offer useful features but add site weight and may not be necessary once you understand on-page optimization fundamentals well enough to audit pages without them.

Key takeaways
  • SEO plugins add code weight that slows site performance, so remove any you are not fully using.
  • Features like sitemaps, schema, and Search Console integration have real value, but the keyword-scoring audit may not improve rankings.
  • Longer SEO descriptions can work better because they are more descriptive and help search engines understand the page.
  • If you are not in the top three positions, the SEO effort is barely worth it, so aim to outperform everyone.
  • Limit plugins to around five or ten maximum and keep back-end code as clean as possible.

Questions this episode answers

Do SEO plugins like Yoast or Rank Math actually help your site rank better?

According to Matt Bertram, the plugins provide useful features such as sitemaps, schema, breadcrumbs, and Search Console integration, but he does not believe entering a target keyword into the plugin itself makes a page rank better. He uses them less and less as he relies on off-page tools and his own process instead.

What is the downside of using SEO plugins on a WordPress site?

SEO plugins add code weight that slows site performance. Matt Bertram moved away from Yoast because it was heavy, and he recommends keeping plugins to around five or ten maximum. He also notes that tools running on the site use resources and can hurt speed, similar to an unused Facebook pixel.

When should a small business owner use SEO plugins?

Matt Bertram compares plugins to training wheels: helpful when you are still learning what to look for on a page. Once you understand on-page fundamentals well, you can audit a page directly and may not need the plugin's scoring algorithm to guide you.

Terms defined in this episode
Crawl budget
The amount of crawling a search engine allocates to a site. The exact budget is unknown, but site owners can send signals via sitemaps and Search Console to suggest which pages to prioritize.
Schema
Structured data markup that SEO plugins like Yoast and Rank Math can add to pages to help search engines understand page content.
Meta keywords
A now largely ignored HTML tag that once told search engines what a page was about. Google has stated it does not use it, and overuse led to spam, as noted in a Yoast article referenced in the episode.
Semantic search
A search approach where engines interpret the meaning and context of content rather than matching exact keywords, influencing how descriptive meta content should be.
