Key takeaways
  • Submit to Google Local Places to appear in the A through G local listings above natural search results.
  • Canonical URL management is primarily about consolidating link juice, not improving bot crawling or user experience.
  • Use listing services like UBL for inbound link juice, but set up your own main listings separately.
  • Submit your site to DMOZ for free high-quality link juice, and to Yahoo Directory for about $299 a year.

Questions this episode answers

How can a local plumber get on the first page of Google ahead of competitors?

The episode recommends first making sure the basics are solid: relevant content, good keywords, and strong backlinks. Then submit to Google Local Places to appear in the local A through G listings above organic results, get active on social media, do article submissions, and build more backlinks.

Does fixing canonical URL issues actually help with Google bot indexing?

According to this episode, the bot does not care about canonical URLs and will index all versions regardless. The real benefit is link juice consolidation. If 400 inbound links are split across four URL versions of the same page, proper canonicalization concentrates all 400 links onto one page. Duplicate content is a secondary concern.

What is header spamming in SEO?

Header spamming is when you place multiple header tags inside a body of content but use CSS to style them so they look like regular body text. The idea is to trick search engines into reading the text as more relevant. The hosts consider it a gray-hat tactic they would not use for client sites.

Are listing services like UBL worth using for local SEO?

The hosts recommend UBL, which costs about $30 a year, primarily for link juice rather than to complete your directory listings. They advise setting up your own main listings separately and using services like UBL as a supplementary source of inbound links back to your website.

Terms defined in this episode
Header spamming
Placing multiple header tags inside body content and using CSS to make them look like regular text, attempting to inflate keyword relevance signals without visible formatting changes.
Canonical URL
The process of consolidating multiple URL versions of the same page, such as with and without www or index.asp, by redirecting them to one preferred URL to concentrate all inbound link juice.
Google Places A through G listing
The local business listings Google displays above natural organic search results, labeled A through G, giving local businesses prominent placement on page one.
Link juice
The SEO value or authority passed to a website through inbound links, which the hosts say is the primary reason to use listing services, directories like DMOZ, and social media pages.
