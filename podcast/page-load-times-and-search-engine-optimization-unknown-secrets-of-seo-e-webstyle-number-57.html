Key takeaways

Questions this episode answers

Does page speed affect Google search rankings?

Yes. Google incorporated page speed into its ranking algorithm. However, the Google Webmaster Central blog noted it is not as important as page relevance. Slow-loading pages cause users to leave quickly, which Google treats as a negative user experience signal.

What free tools can I use to test my page load speed?

The hosts mention four free tools: Google Webmaster Tools under Labs then Site Performance, Page Speed which is an open-source Firefox add-on, YSlow from Yahoo, and Web Page Test. All can give you information about your web page load times.

Should I translate my website if I am targeting customers in another country?

Yes. Google does not automatically translate searches between languages, so a site in Paris should be in French to rank for French searches. The hosts also recommend using a country-code domain like .de for Germany, as it may build more trust with local users.

What causes a web page to load slowly?

According to the hosts, the two main causes are a slow server that cannot serve information quickly, and large files such as oversized images or Flash components. Using CSS and keeping files small are suggested ways to improve load times.
