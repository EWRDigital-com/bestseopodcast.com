Questions this episode answers

What is a 301 redirect and why does it matter for SEO?

A 301 is a permanent redirect using an HTTP error code that tells search engines to pass all link authority from one URL to another. Unlike HTML meta refreshes or code-based redirects without error codes, a 301 transfers Google juice to the target page, consolidating ranking power.

How do I fix Google juice being split between www and non-www versions of my site?

Use a 301 redirect in your .htaccess file to force all traffic to one version. If most inbound links point to the www version, redirect non-www to www. If they are evenly split it does not matter which you choose, since Google passes the juice to the final target page.

Why would a Google Places listing get stuck in needs action or being reviewed status?

According to this episode, the most common cause is having more than one business listing associated with the same address. Google Places allows only one listing per address, and there is no direct support line to resolve it, only a help forum where Google employees may or may not respond.

How did Iranian hackers hijack Baidu's website through register.com?

The hackers used social engineering, calling register.com and persuading a representative to change the email address on the Baidu domain account without proper credentials. This gave them control of the domain and they redirected and defaced the site for a couple of hours. Baidu then sued register.com for gross negligence.

Terms defined in this episode
301 redirect
An HTTP error code that signals a permanent redirect, passing Google juice from the original URL to the target URL, unlike meta refresh or code-only redirects that do not transfer link authority.
Social engineering
A hacking method that relies on manipulating people rather than technical exploits, such as calling a registrar and impersonating an account holder to gain access, as happened in the Baidu domain hijack.
Google juice
Informal term used in this episode for the link authority or ranking power passed between pages via inbound links, which can be split when the same site is reachable at multiple URLs.
.htaccess
A server configuration file used on non-Windows hosting environments that allows 301 redirects and other rules to be applied at the server level without modifying application code.
