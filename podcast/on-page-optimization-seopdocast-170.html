On-Page Optimization - #seopdocast 170

Episode29 min2012-12-06

On-page optimization now requires focusing on user intent, keyword theme building over exact-match repetition, optimized title tags as ads, branding, page speed, and shareable content rather than outdated keyword-stuffing tactics.

Key takeaways
  • Optimize for user intent first, not just keyword rankings, and build your site to handle what users want to accomplish.
  • Use keyword theme building with synonyms and secondary phrases instead of repeating exact-match keywords throughout a page.
  • Write page title tags like advertisements that include a call to action or branding element to improve click-through.
  • Page speed is a ranking signal and a slow site creates a bad user experience that causes bounces.
  • Duplicate content is penalized in competitive markets but may still rank in low-competition small towns where search inventory is thin.

About this episode

On this weeks podcast we talk about incorporating G+ in your web marketing strategy. We then move on to talk about on-page optimization for keywords, search listings and the like. We finally talk about how usability better optimizes your page.

Questions this episode answers

Should I still use exact match keywords on my pages for SEO?

Yes, include the exact match phrase at least once, but the focus should shift to keyword theme building. Use secondary phrases and synonyms that reference the main term so the content reads naturally and avoids being flagged as spammy or panda bait.

Does duplicate content get removed from Google's index?

No. According to this episode, duplicate content is penalized in competitive markets where Google has more results to choose from, but it is not removed from the index. In low-competition small towns with thin search inventory, a duplicate content page may still rank.

How should I write page title tags for better on-page SEO?

Think of the SERP listing as an advertisement. Include your main keyword but also add a supporting call to action or branding element. A title that just repeats the searched keyword with no supporting action gives users no reason to click.

How important is page speed for SEO?

Page speed is a ranking factor but not a dominant one. More importantly, a slow-loading page creates a bad user experience and causes bounces. Google Chrome becoming 26% faster was cited as evidence that users have less patience for slow sites.

Terms defined in this episode
Keyword theme building
A content strategy where a page targets a main keyword by surrounding it with synonyms and secondary phrases rather than repeating the exact match keyword multiple times.
User intent
The goal or task a searcher wants to accomplish when they type a query, which on-page optimization should be designed to satisfy.
Duplicate content penalty
Google deprioritizes pages with duplicate content in competitive search markets with high result inventory, though those pages remain in the index and can still rank in low-competition areas.
SERP listing as advertisement
The idea that a page title tag and meta description should function like an ad, using descriptive language and a call to action to prompt users to click, not just repeat the keyword.
