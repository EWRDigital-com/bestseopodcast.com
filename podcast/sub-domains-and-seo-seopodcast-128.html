Questions this episode answers

Does pointing a domain name to a subfolder instead of the root hurt SEO?

If a server trick is used so the folder path never appears in the address bar, it does not hurt SEO because Google cannot tell the difference. However, if the folder extension is visible in the URL, it may be harmful since Google gives more credibility to the homepage and that is where most inbound links point.

What are the most important tips for optimizing a website for mobile users?

The episode lists nine tips from socialmediaexaminer.com: simplify content repeatedly, plan a minimal page layout, match desktop branding, use white space, avoid Flash and Java, reduce text entry with drop-downs, avoid pop-up windows, use mobile redirects to detect mobile browsers, and always let users switch to the full desktop site.

Why should mobile websites avoid Flash and Java?

Flash is not supported on mobile devices and Adobe announced it would stop developing Flash for mobile and TV. Java also renders poorly on some devices such as BlackBerrys, causing interactive elements like drop-down menus to work inconsistently or fail entirely on smartphones.

Should a business build a separate app or a mobile-optimized website?

According to the hosts, a well-designed mobile website with large icons and a simple layout can replace a dedicated app for most businesses. Apps should be reserved for actual application functionality, not just as shortcuts to a mobile version of a standard site.

Terms defined in this episode
Ziploc algorithm
The hosts' informal name for the Google Freshness update, which rewards websites that produce fresh, regularly updated content.
SEBO (Search Engine Visitor Optimization)
A term used by the hosts to describe analysis of a website from both an SEO and a visitor experience perspective, used alongside traditional SEO analysis.
Mobile redirect
A script placed in a site's header that detects whether a visitor is using a mobile browser and automatically sends them to the mobile-optimized version of the site.
WP Touch
A WordPress plugin mentioned in the episode that enables a website to display a mobile-friendly version of its pages to mobile visitors.
