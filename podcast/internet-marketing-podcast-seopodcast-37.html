Questions this episode answers

Why is animation bad for SEO?

Search engines cannot index information contained inside animation or flash. If a site is entirely flash, Google has no text to read and the site will get no search traffic. The hosts recommend using an HTML version alongside any flash so search engines can crawl the content.

What is SEVO and how is it different from SEO?

SEVO stands for search engine visitor optimization, a term coined by eWebStyle. While SEO focuses on getting traffic to a website, SEVO focuses on converting those visitors into customers through calls to action, forms, and other engagement tools. Traffic without conversion produces no business value.

How should calls to action be used on a website?

Every page should have a call to action, but it does not have to repeat the same phrase each time. It can be a link to a brochure, a portfolio, or a form. The goal is to guide visitors along a path that ends in a purchase, a phone call, or a form submission.

What are the risks of duplicating content from another website?

Duplicating content can expose a site to litigation because copied material is copyrighted. Google also penalizes duplicate content by favoring the original source in search results, meaning the copying site will not rank for that content even if it displays the same information.

Terms defined in this episode
SEVO
Search engine visitor optimization, a term used by eWebStyle to describe optimizing a website to convert visitors into customers through calls to action and engagement, distinct from simply driving traffic.
Duplicate content
Content copied from another source. Google identifies the original owner and will not rank the copying site for that content, and the site owner may also face legal liability.
Splash page
An introductory animated or graphic page shown before the main website. The hosts advise against splash pages because they can hide indexable text from search engines and frustrate navigation.
