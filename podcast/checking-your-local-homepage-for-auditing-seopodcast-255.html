A local homepage SEO audit should check 12 elements including redirects, visible text with NAP, title tags, meta descriptions, mobile rendering, page speed, and social media link accuracy.

Key takeaways
  • Keep homepage redirects to a single 301 and avoid multiple hops to the final URL.
  • Homepage visible text must include business name, category, city name, and NAP information.
  • Use Google PageSpeed Insights to check speed on both desktop and mobile, then act on its recommendations.
  • Tag phone numbers properly so mobile devices recognize and can dial them without guessing.
  • Check social media links for typos and confirm they go to your profiles, not just the platform homepage.

Questions this episode answers

What should be included in a local homepage SEO audit?

The audit should cover redirect structure, visible text including business name and NAP, title tag, meta description, logo optimization, structured local data, phone number formatting, mobile rendering, navigation spider-friendliness, image SEO, page speed, and social media link accuracy, totaling 12 checkpoints.

How should a homepage phone number be handled for mobile users?

The episode recommends tagging the phone number so mobile devices automatically recognize it as a callable number. Vanity numbers should still be accompanied by the real numeric phone number so devices and browsers can interpret and dial it without confusion.

Why does homepage page speed matter for SEO?

Google has confirmed that page speed is a ranking signal. The hosts recommend using Google PageSpeed Insights, which tests both desktop and mobile speed and provides specific recommendations to improve load time, and they describe it as a signal that carries significant weight.

How should social media links on a homepage be set up correctly?

Links should point to your actual social profiles, not the platform homepage. Sidebar icons should be follow links, while end-of-post links should be configured for sharing. The hosts also warn against making social calls to action so prominent that visitors leave your site and never return.

Terms defined in this episode
301 redirect
A permanent redirect used to send visitors and search engines from one URL to another, recommended to be used only once rather than in multiple hops.
NAP
Name, address, and phone number, the core local business contact information that should appear as visible, scannable text on a homepage.
URL tracking
Adding unique identifiers such as UTM parameters to URLs so analytics tools can identify traffic sources, campaigns, and conversions tied to specific links.
Penguin refresh
A re-run of Google's Penguin filter without new ranking signals added, intended to reward sites that cleaned up spammy links and demote those that did not.
