Use Google analytics avoid the flaw - Unknown Secrets of SEO E-Webstyle Number 11

Episode31 min2009-10-19

Google Analytics has a tracking gap for direct file downloads like podcasts and PDFs, and server-side tools like AW Stats or Webalyzer fill that gap by reading raw server log files.

Key takeaways
  • Google Analytics cannot track file downloads made via direct links unless custom tracking code is added to each URL.
  • AW Stats and Webalyzer read server log files to capture downloads that Google Analytics misses entirely.
  • Comparing traffic by week or month rather than day to day makes growth trends easier to spot, especially for seasonal businesses.

Questions this episode answers

Why can't Google Analytics track podcast or PDF downloads from direct links?

When someone clicks a direct link to a file rather than a tracked link on your page, Google Analytics never sees the request. Only the user's computer and your web server know the download happened. Server log tools like AW Stats and Webalyzer capture it because they analyze those server logs directly.

What is the flaw in Google Analytics discussed in this episode?

The hosts describe the flaw as Google Analytics being unable to track file downloads triggered by direct links shared outside the website. If a visitor bypasses the page containing the tracking code and goes straight to the file URL, Google has no way to record that download unless the shared link itself contains tracking code.

How does setting a default page in Google Analytics help with URL tracking?

A bare slash URL and a slash index.asp URL show the same page but register as two separate pages in Google Analytics. By defining a default page for a profile, both URLs are treated as the same page, combining their data so traffic counts are not split across two entries.

What are calls to action and why do the hosts say they matter for SEO?

Calls to action are specific goals you want visitors to take, such as submitting a form, making a purchase, or downloading a file. The hosts argue that driving traffic is only valuable if visitors do what you want, and Google Analytics goal tracking lets you follow the path visitors take toward or away from those actions.

Terms defined in this episode
AW Stats
A server-side statistical analysis tool that reads web server log files to track file downloads and traffic, capturing events that Google Analytics cannot see.
Goal tracking
A Google Analytics feature where you define a target action, such as reaching a thank you page after a purchase, and the tool measures how many visitors complete that action.
Flow or path
A Google Analytics capability that shows the sequence of pages a visitor moves through on a website, revealing where users drop off before reaching a goal.
Default page
A setting in a Google Analytics profile that tells the tool to treat a bare slash URL and a named index page as the same page, merging their traffic data.
