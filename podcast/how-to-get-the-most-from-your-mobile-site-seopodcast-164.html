How to Get the Most From Your Mobile Site - #seopodcast 164

Episode29 min2012-11-02

Mobile SEO requires designing for touch, abbreviation-based search, social sharing, click-to-call, streamlined navigation, and ensuring Google can crawl your mobile subdomain via sitemaps and robots files.

Apple PodcastsSpotifyYouTubeiHeartRadioRSS
Key takeaways
  • Mobile users type abbreviations, so optimize for Google auto-complete phrases rather than full desktop-style keywords.
  • Each mobile page should link clearly to the full desktop site, since some users prefer the complete experience.
  • Click-to-call is essential on mobile pages because the user is already holding a phone.
  • If Google bot is not redirected to your mobile subdomain, include mobile pages in your sitemap so crawlers can find them.
A mobile search is nine times out of ten different from a desktop search or a laptop search.

About this episode

We first go over some basic rules for keeping your website up to date including maintenance and auditing. We then move on to talk about mobile websites including how to streamline content for mobile sites to potentially boost business and keep people happy. Finally, we talk about embracing SEO for mobile sites and how SEO is important for mobile sites.

Questions this episode answers

How should I handle navigation on a mobile version of my website?

Streamline navigation to include only the most important pages. Drop-down hover menus do not work well on touchscreen phones, so reconsider how navigation is presented. Focus on the pages mobile visitors are most likely to need rather than trying to mobilize every page on a large site.

Should I build an app or a mobile-friendly website for my business?

A mobile-friendly site is usually sufficient. An app makes most sense when you need to push information to users, such as new posts, events, or MP3 files. Apps cost significantly more, so determine what the user actually needs before committing to app development.

How do I make sure Google crawls my mobile site?

Google bot does not appear to be a mobile device, so server-side redirects will never send it to your mobile subdomain. Include your mobile pages in your sitemap and ensure the robots file identifies the subdomain so crawlers can reach and index that content independently.

What keyword strategy works best for mobile SEO?

Mobile users rely on abbreviations and Google auto-complete rather than full phrases. Familiarize yourself with how Google auto-fill completes mobile queries in your niche and incorporate those shortened, natural phrases into your mobile page optimization instead of standard desktop keyword forms.

Terms defined in this episode
Click to call
A mobile page feature that lets a user tap a phone number to immediately call a business, highlighted as essential because the user is already holding a phone.
SoLoMo
Social, Local, and Mobile, described in the episode as the three converging priorities that businesses must address in their current marketing strategy.
Robots file
A file used to tell search engine bots where to crawl and what to avoid, which should reference mobile subdomains so Google can discover and index the mobile version of a site.
User agent
The identifier a browser or bot sends to a server; the episode notes that both mobile user searches and search engine bots should reach the same version of a webpage to avoid unintended flagging.
