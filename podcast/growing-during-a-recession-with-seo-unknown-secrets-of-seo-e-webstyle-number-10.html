Questions this episode answers

Why would you want to exclude your office IP address from Google Analytics?

If 50 employees open a browser that defaults to your company website, you get at least 50 visits a day that are internal noise. Excluding the office IP address prevents those visits from skewing your statistics and obscuring real customer traffic data.

What is the session timeout setting in Google Analytics and why does it matter?

The session timeout determines how long Google Analytics waits before ending a visit when a user stops interacting with the server. If your site has a ten-minute video, a short timeout can split one user journey into separate events, making it impossible to trace the full path from arrival to purchase.

How does eWebStyle recommend delivering monthly SEO reports to clients?

eWebStyle prints full-color monthly reports and includes them with the client invoice, rather than sending automated emails. The hosts say this approach reflects their world-class customer service standard because a physical report is more likely to be read than an email.

What is a user agent in the context of website analytics?

A user agent is the browser a visitor was using when they accessed your website, such as Internet Explorer, Mozilla Firefox, or Chrome. Knowing which browsers your visitors use helps you test and confirm your website displays correctly for your actual audience.

Terms defined in this episode
User Agent
Computer terminology for the browser a visitor uses when accessing a website, such as Internet Explorer, Mozilla Firefox, or Chrome.
Session Timeout
The length of time Google Analytics waits without a server interaction before ending a tracked session and starting a new event if the user clicks again.
API (Application Programming Interface)
A programming interface that lets software tap into an external database, such as sending FedEx a zip code and package weight to receive a real-time shipping cost in return.
Call to Action
A specific prompt or destination on a website that you want visitors to reach, which can be tracked in Google Analytics to measure whether users are completing desired steps.
