Sub Domain and Google Link Juice - #seopodcast 119

Episode39 min2011-10-10

Google now classifies subdomain links as internal links in Webmaster Tools, which can reduce apparent external link counts and may affect rankings for sites that used subdomains to inflate inbound link numbers.

Key takeaways
  • Google reclassified subdomain links as internal links in Webmaster Tools, meaning they no longer count as external backlinks.
  • Sites using proper 301 redirects from subdomains likely see little ranking impact since link juice already consolidated.
  • Google has not publicly stated how subdomain link weighting changes the ranking algorithm, so the full effect is unknown.
  • Social media ROI can be tracked through analytics by following the funnel path from a social platform to a checkout page.
  • Creating a social media profile and abandoning it is a waste of time and blocks others who might actually use the username.

Does Google treating subdomain links as internal links hurt SEO?

It can hurt sites that relied on subdomain links appearing as external backlinks to inflate their inbound link count. Sites with proper 301 redirects from subdomains are less likely to be affected because their link juice was already consolidated to one location.

Terms defined in this episode
Subdomain
A prefix before a root domain, such as www.yourdomain.com or jobs.yourdomain.com. Google previously counted links from subdomains as external links but now classifies them as internal links in Webmaster Tools.
301 Redirect
A permanent redirect that Google understands, used to point one URL to another so that link juice consolidates to a single destination.
Link Juice
The ranking value or authority passed through a link from one page or domain to another, referenced here in the context of subdomain links now being treated as internal rather than external.
