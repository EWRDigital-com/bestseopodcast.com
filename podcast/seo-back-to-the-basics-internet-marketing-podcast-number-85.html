Key takeaways
  • Use title tags and description tags with keywords, not just a company name, on every page.
  • Target keyword density between 1% and 3% and write content a real person would want to read.
  • A site can have four duplicate homepages; a 301 redirect consolidates link juice into one.
  • Alt tags, image file names, and link titles are all additional places to place keywords without stuffing.
  • Calls to action should tell visitors explicitly what to do and give a specific reason to act.

Questions this episode answers

Why does having www and non-www versions of a site hurt SEO?

If both versions resolve without a redirect, Google treats them as separate pages with duplicate content. Links pointing to different versions split link juice across multiple pages. A 301 redirect consolidates all versions into one, combining all inbound links into a single page with full link authority.

What keyword density should website content target?

The hosts recommend targeting between 1% and 3% keyword density. They caution that content writers can get overzealous at 3%, so they often say 1%. The goal is keyword-rich content that reads naturally, not stuffed repetition that sounds awkward and signals spam to Google.

How do alt tags and image file names help with SEO?

Google cannot read images, so alt tags tell search engines what an image contains and are a valid place for keywords. Naming the image file itself, for example Houston-Plumber.jpg instead of image84962.jpg, adds another relevant signal Google uses when indexing images in search results.

When is a lower time on site actually a good thing?

For service-based businesses like plumbers or doctors, a decreasing time on site paired with rising conversions means visitors are finding the call to action quickly and contacting the business. The hosts say this is the desired outcome, though informational or video-heavy sites still benefit from longer visits.

Terms defined in this episode
301 redirect
A permanent redirect that forwards one URL to another, consolidating duplicate homepage versions so all inbound link juice flows to a single page.
SEVO (Search Engine Visitor Optimization)
A concept the hosts use meaning optimizing a site for both search engines and human visitors, emphasizing readability, lists, and calls to action over pure keyword placement.
Contextual relevancy
Google Caffeine checks whether a keyword belongs in a given sentence, effectively reading content like a person to judge whether density is appropriate and whether the word fits the context.
Link title
An HTML attribute on a link that displays text when a user hovers over it; search engines index it as an additional keyword placement, and screen readers use it to describe the link destination to disabled users.
