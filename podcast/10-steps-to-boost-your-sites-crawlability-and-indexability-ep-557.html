Key takeaways
  • If Google does not index a page or a link pointing to your site, it provides zero SEO benefit.
  • Internal linking is one of the biggest on-site actions you can take to guide Google and users to important pages.
  • A robots.txt file misconfigured with a no-index directive can cause rapid and severe ranking drops.
  • Each redirect hop can cost roughly 15% of link equity, so minimize chains and fix multi-hop redirects at the source.
  • Pages below a 90% indexability rate signal issues worth investigating in Google Search Console.

Questions this episode answers

What is the difference between crawlability and indexability?

Crawlability refers to how well search engine bots can scan and access your webpages. Indexability measures the search engine's ability to analyze your website and add pages to its index. Both must work for content to appear in search results.

How do you check if your site has indexability problems?

In Google Search Console, calculate your indexability rate by dividing the number of pages in Google's index by the total pages on your site. If that rate falls below 90 percent, the episode recommends investigating further using tools like Screaming Frog.

Why is robots.txt so important for crawlability?

A robots.txt file in your site's root directory tells crawlers which pages to access or skip. A misconfigured file, such as one that blocks the entire directory, can cause Google to stop indexing your site entirely, leading to rapid ranking drops.

What is IndexNow and how does it help with indexability?

IndexNow is a relatively new open source protocol that allows website publishers to instantly submit URLs simultaneously across participating search engines via an API. It updates search results based on the latest content changes and requires generating an API key hosted in your directory.

Terms defined in this episode
Crawlability
How well search engine bots can scan and access webpages on a site. Poor crawlability from broken links or dead ends prevents content from being indexed.
Indexability Rate
The number of pages in Google's index divided by the total number of pages on your website. A rate below 90 percent indicates problems to investigate.
IndexNow
An open source protocol that allows website publishers to instantly index URLs across participating search engines via an API, updating results based on the latest content changes.
Canonical Tag
An HTML tag that consolidates signals from multiple URLs, telling Google which version of a page to index while skipping duplicates and outdated versions.
