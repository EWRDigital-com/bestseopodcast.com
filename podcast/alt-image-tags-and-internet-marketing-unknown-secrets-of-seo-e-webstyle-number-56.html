Questions this episode answers

What is an alt tag and why does it matter for SEO?

An alt tag is attached to an image and contains text that search engines can read. Because search engines cannot read images directly, the alt tag tells them what the image is about. The hosts recommend including keywords in alt tags to help search engines understand page content.

Why should every page have a unique title tag?

Google uses title tags to understand what a page is about. Using the same title, like a company name, on every page does not signal any specific topic to Google. Each page should have a unique title tag with a keyword relevant to that specific page.

What is SEVO and how is it different from SEO?

SEVO stands for search engine visitor optimization. The hosts coined this phrase to describe optimizing a website for visitors who arrive from search engines. It includes having the searched keywords visible on the landing page, clear calls to action, and aesthetically pleasing design so visitors do not immediately leave.

Why should internal links not all point to the home page?

Linking every page back to the home page concentrates page rank there and means search engines will likely display the home page for many queries. If the home page lacks the specific keyword content a visitor searched for, they will bounce. Links should point to the most relevant page for each topic.

Terms defined in this episode
Alt tag
A text tag added to an image element that search engines read in place of the image itself, since search engines cannot read image files.
SEVO
Search engine visitor optimization, a phrase coined by eWebStyle to describe optimizing a website so visitors arriving from search engines quickly find relevant content and take action.
Anchor text
The clickable text in a hyperlink. Using keyword-rich anchor text tells Google what the destination page is about, which influences how that page ranks.
Dangling page
A page on a website that exists but is not linked from anywhere within the site, making it invisible to search engine crawlers.
