Key takeaways
  • Run metatags.org on every page of your site, not just the homepage, to catch errors like stop words in keyword tags.
  • Title and description tags should differ on every page and target the specific keywords relevant to that page's content.
  • Your meta description tag is your first sales pitch in search results, so write it as a short, targeted pitch to searchers.
  • Use keyworddensity.com to compare your keyword density against a competitor ranking on the first page of Google.
  • Cloaking, showing search engines a different page than visitors see, can get your website banned by Google.

Questions this episode answers

What is a keyword density tool and how do you use it for SEO?

A keyword density tool like keyworddensity.com shows how often a keyword appears relative to total words in your title, meta description, meta keywords, and visible text. You compare your page against a competitor ranking on page one to find where your keyword usage falls short.

Which meta tags does Google actually care about?

According to this episode, search engines focus primarily on the keywords tag, the description tag, and the title tag. The title tag carries significant weight in rankings and also appears in the browser bar, while the description tag often controls what searchers see in results.

What is a stop word in SEO and why does it matter?

A stop word is a common word that most search engines ignore. In this episode, the word 'it' was flagged as a stop word inside a keyword tag, meaning 'IT' as in Information Technology was being read as the word 'it' and ignored, which the metatags.org tool caught automatically.

What is cloaking and why should you avoid it?

Cloaking means your server detects when a search engine bot visits and serves it a different version of the page than regular visitors see. Google can detect this and will ban your website, and according to this episode, getting unbanned is not easy.

Terms defined in this episode
Keyword Density
The number of times a target keyword appears on a page divided by the total word count, expressed as a percentage, measured across title, meta description, meta keywords, and visible text.
Meta Tag
Code placed behind the scenes of a webpage, not visible to visitors but read by search engine bots, that can convey keywords, description, title, author, and other page information.
Cloaking
A technique where server code detects search engine bots and serves them a different version of a webpage than human visitors see, which violates Google guidelines and can result in a site ban.
301 Redirect
A permanent redirect that passes the existing SEO value, called Google Juice in this episode, from an old URL to a new one so rankings are not lost when a page name changes.
