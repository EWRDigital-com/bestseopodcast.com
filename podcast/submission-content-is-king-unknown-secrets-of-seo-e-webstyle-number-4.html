Submission, Content Is King - Unknown Secrets of SEO E-Webstyle Number 4

Episode31 min2009-10-01

Content is king in SEO because search engine spiders read text, not images, so websites need relevant, dynamic, keyword-rich copy plus manual search engine submission and RSS feeds to rank well.

Apple PodcastsSpotifyYouTubeiHeartRadioRSS
Key takeaways
  • Search engine spiders cannot read text inside images, so keywords must appear as actual page text to rank.
  • Dynamic content from RSS feeds adds fresh, relevant text that search engines favor over static pages.
  • SEO copywriting means writing website text with target keywords in mind, not just describing a business like a brochure.
  • You only truly need placement on Google, Yahoo, and MSN. Submissions to a thousand directories add little value.

About this episode

We cover Google Analytics and how Google Analytics can guide your websites SEO efforts in the right direction for ROI. Directory submission, find out what is the value of directory submission and should you be submitting. RSS stands for real simple syndication, what is it and how can it help your website.

Questions this episode answers

Why is content called king in SEO?

Search engine spiders crawl website text and store it so they can match it to search queries. If a page uses images instead of text for key terms, or omits relevant keywords entirely, the site will not appear in results. Sufficient, relevant text is what gives a page credibility with search engines.

What is RSS and how does it help SEO?

RSS stands for Real Simple Syndication. By placing an RSS feed on a webpage, a site owner can pull in fresh, topic-relevant articles automatically. Each new article updates the page content, giving search engines dynamically changing, relevant text to crawl, which search engines favor over static pages.

What is SEO copywriting and why does it matter?

SEO copywriting means writing website text so that target keywords appear naturally and repeatedly in the copy. A site that describes a business in brochure language without naming the service, location, or owner misses the keywords spiders look for. Professional SEO copywriters craft content that matches what search engines and searchers expect to find.

Do I need to manually submit my website to search engines?

Not always. If your site is linked from an already-indexed page, crawlers will find it. However, a brand-new site with no inbound links will not be discovered until it is submitted manually. The episode recommends submitting to Google, Yahoo, and MSN yourself rather than using automated bulk-submission services.

Terms defined in this episode
RSS Feed (Real Simple Syndication)
A standard format that syndicates text content so it can be automatically pulled onto other websites or delivered to programs like iTunes, keeping page content fresh and dynamically updated.
Bounce Rate
A Google Analytics metric showing how often visitors leave a website without taking any action. A lower bounce rate indicates visitors are engaging with the site.
SEO Copywriting
Writing website text with deliberate placement of target keywords so search engine spiders can index the page for relevant queries, as opposed to brochure-style writing that omits those terms.
Call to Action
An element on a webpage that prompts a visitor to purchase, call, or email. The episode recommends having one on each and every page.
