Key takeaways
  • Link building quality and a diverse link profile matter more than link quantity alone.
  • Buying paid ads has no direct correlation to improving organic search rankings.
  • Videos and social media help SEO but neither is sufficient on its own.
  • Rewriting manufacturer product descriptions helps e-commerce sites avoid duplicate content penalties.
  • SEO is the foundation of a website but must be combined with social, paid search, and newsletter marketing.

Questions this episode answers

Do paid ads help organic search rankings?

According to Bing's Dwayne Forrester and the hosts, there is no correlation between paid ads and organic rankings. However, running paid ads can reveal which keywords convert best, and that data can then be used to optimize a site organically.

What is SEVO and how does it relate to SEO?

SEVO stands for Search Engine Visitor Optimization, a term the hosts use to describe optimizing for the user experience alongside traditional SEO. They argue usability and rankings are now closely connected, and videos are cited as one tool that improves on-site engagement.

Terms defined in this episode
Link profile
The full set of inbound links pointing to a site, ideally diverse in anchor text, sources, and type, and free from spammy or paid links.
Schema markup
Code added to page content that helps search engines index, parse, and display information, such as enabling carousel or image display in results.
SEVO
Search Engine Visitor Optimization, a term used by the hosts to describe optimizing a site for both search engines and the user experience simultaneously.
Duplicate content
Identical or near-identical text appearing on multiple pages or sites, a common issue for e-commerce stores using unmodified manufacturer product descriptions.
