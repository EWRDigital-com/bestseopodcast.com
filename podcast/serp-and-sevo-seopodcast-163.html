Key takeaways
  • Meta descriptions are your first pitch to searchers on the SERP, so use the full character limit with a clear offer.
  • Title tags should lead with the target keyword, not the company name, for example 'water heater repair slash Joe Blow plumbing'.
  • A custom 404 page can fire an email alert so overlooked high-ranking URLs get rebuilt fast after a site redesign.
  • Recurring content on the home page should be paired with static, keyword-rich introductory text that never changes.
  • Always check the robots.txt file on a new project because a previous designer may have excluded the entire site from Google's index.

Questions this episode answers

How should title tags be written for better SEO?

The biggest mistake is using only the company name in every title tag. Instead, lead with the main service or keyword for that page, for example 'water heater repair slash Joe Blow plumbing', so the title is descriptive and keyword rich.

Why is a custom 404 page important for SEO?

During a site redesign, pages that ranked well can be dropped or overlooked. A custom 404 keeps visitors on your site and, with the right code, can fire an email alert so you know which lost URLs need to be rebuilt quickly.

How does site speed affect SEO and user experience?

A slow site causes visitors to bounce. A common problem is clients uploading oversized image files, such as a 1.5 megabyte file at 50 by 50 pixels. Resizing images before uploading and using WordPress compression helps keep load times fast.

Terms defined in this episode
SERP
Search engine results page, where your meta description appears and serves as a first chance to pitch your company to potential visitors.
Canonicalization
Server-level configuration that ensures non-www URLs redirect to www versions, and that pages with strong rankings are preserved through redirects during a site relaunch.
XML Sitemap
A file submitted to Google Webmaster Tools so Google can receive site updates and index pages the way you intend.
Robots.txt
A file that signals to the Google bot which pages to index; a misconfigured file can exclude an entire website from Google's index.
