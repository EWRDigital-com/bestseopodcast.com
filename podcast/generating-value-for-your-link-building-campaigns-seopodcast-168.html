Key takeaways
  • Focus link building on high-PR, credible sites that require manual approval, not automated mass submissions.
  • Balance anchor text so it does not appear entirely keyword-rich, since natural link profiles include varied text.
  • Create link-worthy content like infographics, white papers, videos, and guest posts to earn links organically.
  • Monitor your link profile in Webmaster Tools and use the disavow feature to remove low-quality incoming links.
  • Build personal relationships at industry events to earn high-value, topically relevant links that Google trusts.

Questions this episode answers

What link building strategies work after Google Panda and Penguin updates?

The episode covers ten strategies: identify viable link opportunities, use tools like Open Site Explorer and Majestic SEO, target sites with manual approval, win the content marketing game, use balanced anchor text, avoid mass link building tactics, use social media, monitor your link profile, analyze competitor links, and build personal relationships.

Why is balanced anchor text important for link building?

Google recognizes that a real, natural link profile does not use all keyword-rich anchor text. Some links say 'website,' some say 'click here,' and some use the domain name. Using only keyword-rich anchor text looks artificial and can trigger algorithm penalties.

How do you monitor and clean up a bad link profile after a Penguin penalty?

The hosts recommend monitoring which sites link to yours, checking for low-quality or low-PR inbound links, and using the disavow feature in Google Webmaster Tools to remove harmful links. Analyzing why high-quality links are shared and repeating that strategy is also advised.

Why should you avoid mass link building services?

Services that submit your site to thousands of links for a low fee produce links with no weight or juice. Google recognizes the artificial pattern, and because everyone uses these services, those links carry no real value and can damage your site's standing.

Terms defined in this episode
Link Profile
What Google sees regarding how your links are structured, including where inbound links come from, where outbound links go, when you got them, how long you have had them, and whether the servers are secure.
Penguin Update
A Google algorithm update that targeted low-quality and low-value links, causing ranking declines for sites with poor link profiles.
EMD Update
A Google algorithm update, short for exact match domain, designed to reduce the ranking value given to keyword-rich domain names.
SEVO
Search engine visitor optimization, referring to what you do with visitors after they arrive on your website following a search engine result page visit.
