Key takeaways
  • Keyword-focused title tags should place the company name last, or omit it entirely unless the brand is a household name.
  • Fresh content gained even more ranking value after Panda, and sharing it on social platforms like Twitter and Facebook amplifies that value.

  • Getting a link from a quality, frequently updated site gets a new site indexed faster than submitting through Google's standard submission form.
  • PPC managers need SEO expertise because Google evaluates the quality of landing pages tied to paid campaigns.

Questions this episode answers

Why do canonicalization issues hurt SEO rankings?

Google treats www and non-www versions of a domain, plus URLs with and without index.html, as separate pages. Inbound links spread across those variations divide link juice. A 301 redirect consolidates all versions to one preferred URL so Google attributes all link equity to a single page.

Terms defined in this episode
Canonicalization
The problem of multiple URLs representing the same page, such as www and non-www versions, which splits link juice. A 301 redirect resolves this by pointing all versions to one preferred URL.
301 redirect
A permanent redirect that tells Google one URL should replace another, consolidating link equity to a single authoritative page.
Link juice
The ranking value passed through inbound links. Splitting links across multiple versions of the same URL reduces the value each version receives.
SIVO
Search Engine Visitor Optimization, a term used by the hosts to describe their website analysis approach.
