Questions this episode answers

What makes SEO content effective enough for people to share it?

According to the hosts, content should be informative, captivating, and entertaining. When content is entertaining, people pay attention, learn something, and are more likely to share it. Content that does not help users will not be shared, and bounce-backs can signal poor quality to Google.

Why is having a great website so important before running SEO or PPC campaigns?

The hosts explain that the website is the foundation of any internet marketing campaign. They give an example of a client whose sales increased after a redesign even before a new PPC campaign started, showing that traffic driven to a poorly designed site will not convert.

What SEO basics do site owners most commonly overlook?

The episode lists robots files, alt tags on images, updated XML sitemaps, correct 301 redirects, and avoiding duplicate title tags and meta tags as basics that people forget, often because they rush to launch and plan to fix things later but never do.

How can CSS absolute positioning help SEO without being black hat?

By using CSS absolute positioning, keyword-rich content can appear first in the page code so Google treats it as most important, while it can display anywhere visually in the browser. Since the same content is shown to both Google and users, this is not considered black hat.

Terms defined in this episode
robots file
A file that tells Google bot and other search crawlers which pages on a site to index and which to exclude.
301 redirect
A permanent redirect used to send users and search bots from an old URL to a new one, preserving page authority when URLs change.
XML sitemap
A file submitted to webmaster tools that lists a site's pages so search engines can find and index them, and should be updated whenever navigation or pages change.
top heavy algorithm
A Google algorithm change that places the most relevance on content appearing at the top portion of a page's code.
