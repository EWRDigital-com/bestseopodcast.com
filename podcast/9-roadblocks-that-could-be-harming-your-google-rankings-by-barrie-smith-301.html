9 Roadblocks That Could be Harming Your Google Rankings by Barrie Smith #301

Ep. 30137 min2015-12-17

Nine roadblocks harm Google rankings, including duplicate content, thin content, anchor text over-optimization, bad backlinks, non-optimized page titles, insecure sites, accidental no-indexing, poor user experience, and using a bad agency.

Apple PodcastsSpotifyYouTubeiHeartRadioRSS
Key takeaways
  • Duplicate or thin content triggers a Panda penalty, so write original, meaty content and fix canonicalization issues on tag-generated URLs.
  • Over-optimizing anchor text looks unnatural to Google, so mix keyword anchors with click-here text, company names, and raw domain links.
  • Bad backlinks trigger a Penguin penalty, so link out to relevant suppliers and research sources rather than chasing any link available.
  • HTTPS is a confirmed ranking factor and adds user confidence, especially for WordPress sites and any site collecting visitor information.
  • Check your robots.txt file to make sure your site is not accidentally blocking Google from indexing your content.
search ranking can only get you so far. If your user experience is bad, it doesn't matter how many visitors your site gets.

Questions this episode answers

What is the Panda penalty in SEO?

The Panda penalty is applied by Google to sites with duplicate, thin, spun, or irrelevant content. The episode explains it is triggered when content is not original, not useful, or copied from elsewhere, and it prevents the site from ranking well in search results.

Does site security affect Google rankings?

Yes. The episode states Google announced that site security is a ranking factor, with HTTPS sites receiving a ranking benefit. A secure server also builds user confidence and adds a layer of protection, especially for WordPress sites that make up about 20 percent of all websites.

How can a site accidentally get no-indexed?

A robots.txt file can be configured to tell Google to ignore an entire site. The episode describes a real client whose site was fully disallowed in their robots file, meaning no SEO work would ever produce results until that file was corrected.

Terms defined in this episode
Panda penalty
A Google algorithm penalty targeting sites with duplicate, thin, spun, or irrelevant content that provides poor user value.
Penguin penalty
A Google algorithm penalty that targets manipulative link profiles, including paid links, link swapping, and link spamming.
Anchor text over-optimization
Using keyword-rich anchor text excessively in links, which Google now penalizes as unnatural; a healthy link profile mixes keywords with brand names, domain names, and generic text like click here.
Robots.txt
A file on a website that instructs search engine crawlers which pages or folders to index or ignore; misconfiguring it can accidentally block an entire site from Google.
