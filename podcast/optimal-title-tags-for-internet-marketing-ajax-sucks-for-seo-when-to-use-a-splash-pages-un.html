Key takeaways
  • Use descriptive anchor text instead of 'click here' so Google understands the destination page's topic.
  • Ajax-loaded content is invisible to search engine bots, so avoid it for any content you want indexed.
  • Splash pages hurt SEO and user experience; only use them when there is real utility, such as country or language selection.
  • Title tags should be unique per page and kept under 65 characters to avoid keyword truncation.
  • Always add image alt attributes because Google counts them and they support both SEO and user experience.

About this episode

We discuss optimal title tags for SEO. We talk about the example of Internet Marketing Houston. Many people what to have fancy FLASH splash pages, they are beautiful but when should you use them and when should you avoid them. We also discuss some negatives like, AJAX sucks for SEO.

Questions this episode answers

Why is Ajax bad for SEO?

Search engine bots like Google and Yahoo do not click dynamically loaded links or read Ajax-generated content. If your target keywords only appear in Ajax-loaded sections, Google will not know that content exists on your page, so you cannot rank for those terms.

How long should a title tag be for SEO?

Title tags should be about 65 characters. Going over that limit causes the text to get cut off, which can chop off important keywords. Each page should also have a unique title tag reflecting that page's specific topic rather than a generic site-wide name.

When is a splash page acceptable for SEO?

A splash page is acceptable when it serves genuine utility, such as country or language selection for a global site. FedEx is cited as an example where users select their country before entering the site. Otherwise, splash pages hurt both SEO and user experience and should be avoided.

Why should image alt attributes be used for SEO?

Google counts image alt attributes when indexing a page. Adding descriptive alt text to images provides an additional SEO signal and also improves usability by displaying text when a cursor hovers over an image, making it a practical double benefit for site owners.

Terms defined in this episode
Ajax
A technology that dynamically brings content onto a webpage without refreshing the entire page, which prevents search engine bots from reading that content.
Alt attribute
Text added to an image tag that appears when a cursor hovers over an image; Google counts it as an SEO signal for the page.
Friendly URL
A readable URL such as patrickwanus.com/quit-smoking rather than a dynamic URL with database query strings; preferred by Google and easier to share.
Title tag
An HTML tag that labels a webpage; it should be unique per page, reflect that page's keywords, and stay within approximately 65 characters to avoid truncation.
