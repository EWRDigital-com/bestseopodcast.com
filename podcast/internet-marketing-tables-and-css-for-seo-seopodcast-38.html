Internet Marketing, Tables and CSS for SEO - #seopodcast 38

Episode29 min2009-11-30

CSS and divs outperform tables and inline font tags for SEO because they reduce page load time, keep code clean, and make it easier for search engine spiders to crawl your site.

Key takeaways
  • Use CSS cascade style sheets instead of inline font definitions to reduce page load time.
  • Avoid table-based layouts because some browsers delay rendering until the full table loads.
  • Validate your HTML to catch dead links and unclosed tags that hurt crawlability.
  • Use the strong tag instead of bold, and em instead of italics, as the older tags are outdated.
  • After getting traffic to your site, focus on converting visitors, what eWebStyle calls search engine visitor optimization.

Questions this episode answers

Are tables bad for SEO?

According to this episode, there is no general consensus that tables are definitively bad for SEO, but they do have drawbacks. Some browsers will not show any content in a table until the whole table has loaded, which can hurt user experience and slow rendering for visitors and crawlers.

Why should I use CSS instead of inline font tags on my website?

Using a cascade style sheet means you define font type, size, and color once and reference it throughout the page, rather than repeating that code on every line. That shorter, less redundant code reduces page file size and makes the page load faster, which benefits both users and search engines.

What is search engine visitor optimization?

The hosts describe search engine visitor optimization, a term they say they coined at eWebStyle, as the practice of converting website visitors into actual business value after you have already brought them to your site, addressing calls to action and what you do with the traffic you receive.

Terms defined in this episode
Divs
HTML division elements that can show or hide sections of content on a page, useful for forms, text, and images, and readable by search engines.
Cascade Style Sheet (CSS)
A separate style sheet where font type, color, size, and background colors are defined once and referenced throughout the page, reducing code repetition and speeding up page load.
Validate
The process of checking that a website has correct open and closed HTML tags, no dead links, and properly structured code so search engines can crawl it without errors.
Search Engine Visitor Optimization
A term coined by eWebStyle referring to the practice of converting site visitors into business value through calls to action after driving traffic via SEO.
