Key takeaways
  • Hiding white text on a white background or using CSS to push text off-screen violates Google's policies and risks a site ban.
  • Google's robots flag hidden text automatically without judging relevance, so even relevant hidden keywords can trigger a penalty.
  • Kay Bailey Hutchison's site was banned from Google for using CSS to hide a block of text behind an image.
  • A Lynx Browser shows a website as plain text, revealing exactly what Google's crawlers can and cannot see.

Questions this episode answers

Does hiding text on a website help with SEO or will it get you penalized?

Hiding text is a black-hat SEO technique that violates Google's policies. Even if the hidden keywords are relevant, Google's automated robots flag the hidden text without judging its relevance, and the site can be banned. A competitor reporting the practice to Google can also trigger a manual penalty.

What happened to Kay Bailey Hutchison's website and why?

Kay Bailey Hutchison's site was banned from Google because it used CSS, also called cascade style sheets, to hide a large block of text behind an image. One of the hidden phrases was 'why do we have knees,' which was irrelevant to her site's content.

What is a Lynx Browser and why should website owners use one?

A Lynx Browser displays a website as plain text with no images, showing exactly what Google's crawlers see. If the text view shows nothing relevant, it explains why a site may not be indexed. The hosts recommend visiting lynx.isc.org or searching for Lynx Browser to find a tool.

In their words

If you had all images and you hadn't been listening to our podcast and you didn't know that you shouldn't have all images because Google can't see it, you could pull up your website in a text browser and you would see nothing.

Terms defined in this episode
Hidden text
Text made invisible to website visitors, such as white text on a white background, used to insert keywords without displaying them to users. Google classifies this as a black-hat technique and can ban sites that use it.
CSS image replacement
A technique where CSS is used to indent text thousands of pixels off-screen and an image is placed over it, effectively hiding the text from users while keeping it in the HTML source code.
Lynx Browser
A text-only browser that strips all images from a webpage, showing only the text content. It reveals what Google's crawlers see when they index a site.
Black hat SEO
SEO techniques that violate search engine guidelines, such as hiding text, which can result in a website being banned from Google's index.
