Questions this episode answers

What is first link priority in SEO?

First link priority means Google only counts the first link on a page that points to a given destination. If the same page is linked multiple times, Google ignores the duplicates, so the anchor text and placement of that first link matters most for passing value.

How should I structure my site navigation to take advantage of first link priority?

In your code, tell the crawler to skip your top navigation and go to your secondary nav first. Your secondary nav typically contains keyword-rich links like product or service pages, which carry more SEO value than generic labels like home or about us.

Does using no-follow on links still redistribute extra page rank to other links on the page?

No. Previously, no-following some links would concentrate page rank on the remaining links, giving them a larger share. That changed with an algorithm update, so no-following links no longer increases the page rank flowing to the other links on the page.

When is the best time to post on Twitter and Facebook for maximum engagement?

According to this episode, the best time to get retweeted on Twitter is between 3 p.m. and 6 p.m., and the best day is Friday. For Facebook, the best time to share content is 9 a.m. and the best day is Saturday.

Terms defined in this episode
First Link Priority
Google's behavior of counting only the first link on a page that points to a given destination, ignoring subsequent duplicate links to the same page.
Link Sculpting
The practice of controlling how page rank flows through a page by managing which links are followed or no-followed and which links appear first in the page code.
Anchor Text
The clickable, keyword-rich text of a hyperlink, which carries more SEO value than image alt tags when Google evaluates links.
Inbound Marketing
A strategy of getting clients to come to you through SEO and social media rather than outbound methods like cold calling, TV, or yellow pages.
