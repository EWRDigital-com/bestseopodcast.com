Keeping Track of Site Metrics - Internet Marketing Podcast Number 195

Episode18 min2013-07-29

Tracking site metrics like load time, page speed, Google Analytics integration, XML sitemaps, and dead link checks are essential parts of a complete website audit that many site owners overlook.

Key takeaways
  • Run SEO audits frequently, especially on sites with multiple people managing content, to avoid algorithm penalties.
  • Load time and page speed are two separate metrics; both must be optimized to improve rankings and user experience.
  • Multiple analytics codes on a site can cause broken conversion tracking; audit and clean all scripts regularly.
  • Use Xenu to scan your site for dead links and remove or redirect any 404s you find.
  • Always disavow bad or sketchy inbound links to prevent unwanted associations that can hurt your site.

Questions this episode answers

What is the difference between load time and page speed in SEO?

Load time is how long it takes a specific element like an image or download to load after a click. Page speed is how fast the entire page renders on a click, including all media and load time. Both should be as fast as possible to improve Google rankings and user experience.

Should I include old 301 redirected pages in my XML sitemap?

No. Old pages that have 301 redirects should be removed from your sitemap. The sitemap is a tool for Google and Bing to know which pages are active and live. Only include your new, live pages so search engines crawl the correct URLs.

Why is Google Analytics integration so important to audit?

When multiple companies manage a site over time, duplicate or incorrect analytics codes can end up on the site, causing conversion tracking to break or report incorrectly. Auditing ensures you have the right tracking codes, in the right place, with no duplicates or outdated scripts.

Terms defined in this episode
Load Time
How long a specific element such as an image or downloadable file takes to load after a user clicks, distinct from overall page speed.
Page Speed
How fast an entire page renders on a click, including all media and load time; a factor in Google page rank and placement.
Xenu
A tool spelled X-E-N-U used to scan a website and generate a report identifying which links are dead and which are working.
301 Redirect
A redirect used to preserve link juice from old pages and pass it to new pages, keeping SEO value while pointing users and crawlers to the current URL.
