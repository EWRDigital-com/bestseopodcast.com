#SEOPodcast292 - Eight Most Common but Simple Missed Opportunities in Optimizing Content

Episode37 min2015-10-07

Eight common missed SEO opportunities include duplicate page titles, weak meta descriptions, ignored site links, missing structured markup, broken analytics, poor video embedding, extraneous code, and no XML sitemap.

Key takeaways
  • Keep page title tags around 55 characters and use one primary keyword per page, adding location if relevant.
  • Use your best-converting paid ad copy as your meta description to improve click-through from search results.
  • Mark up NAP, products, reviews, and events with structured data to generate rich snippets and aid crawlers.
  • Remove unused plugins and extra code to improve page speed, site security, and search rankings.
  • Update your XML sitemap and resubmit it in Search Console every time you add or change content.

Questions this episode answers

Why do duplicate or poorly written page title tags hurt SEO?

Duplicate titles waste each page's chance to target a unique keyword and fail to earn the click. Hosts recommend staying around 55 characters, using one primary keyword, adding a location identifier if relevant, and only including the company name if space allows.

What is the fastest way to get new content indexed by Google?

Maintain an XML sitemap registered through Search Console. The article cited in the episode found that a new post appeared in Google's index in eight seconds when the sitemap was in place, though the hosts note frequent content publishers get faster recrawls than infrequent ones.

Why is broken or misconfigured analytics a serious SEO problem?

Without properly firing analytics you cannot benchmark organic traffic, track conversions, or make educated decisions about where to invest. The hosts recommend filtering out internal IP addresses, spam bots, and ghost visits, and setting up goals for every conversion type including phone calls.

Terms defined in this episode
Structured markup
Code added to website elements such as addresses, events, products, and reviews to help search bots understand content and generate rich snippets in search results.
Rich snippets
Enhanced search result listings produced by structured data markup that make a SERP result larger and more visible, as described in the episode.
CVO (Search Engine Visitor Optimization)
What you do with a visitor after they arrive from a search engine, which the hosts say begins at the SERP itself.
XML sitemap
A file submitted through Search Console that tells search engines about new or updated pages so they can be crawled and indexed more quickly.
