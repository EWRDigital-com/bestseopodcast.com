Do Google AdWords ads affect organic search rankings?

No. According to the hosts, Google AdWords and organic search run on two completely separate algorithms with no effect on each other. They recommend running both simultaneously, using PPC for immediate ROI and keyword data while SEO work is underway.

Terms defined in this episode
Long tail keywords
Phrases that are uncompetitive and less searched but attract highly qualified visitors who are typically in purchasing mode and know exactly what they are looking for.
Soft 404
A situation where a website server is configured to send the wrong response code to a Google spider, which Google says it has improved its ability to detect.
Unordered list
An HTML bulleted list that Google values because users prefer reading bullet points, making it a content element that can add ranking value to a page.
PPC (Pay-Per-Click)
A paid advertising model, such as Google AdWords, that operates on a completely separate algorithm from organic search and has no effect on organic rankings.
