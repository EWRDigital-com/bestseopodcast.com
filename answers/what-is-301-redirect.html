What is 301 Redirect?

Answered across 30 episodes of The Best SEO Podcast

Edited and curated by Matthew Bertram, host of The Best SEO Podcast (produced by EWR Digital).

The short answer

A technical instruction that maps old web addresses to new ones during a domain or website migration, preserving accumulated search traffic and rankings.

How experts on the show define it

A technical instruction that maps old web addresses to new ones during a domain or website migration, preserving accumulated search traffic and rankings.

From Unlocking Transformative Rebranding: Strategies and Trends with Jim Heininger

A permanent redirect used to point one domain or page to another, discussed in the episode as a way to consolidate an inherited domain with 28,000 links into an existing site.

From Content Redesign Tips - #SEOpodcast 244

A permanent redirect that sends users and search engines from an old URL to a new one, preserving link equity and preventing 404 errors during a site relaunch.

From New Website? Do an SEO Audit! by Janet Driscoll Miller #357

A permanent redirect from one URL to another, used when a page moves to a new location, as opposed to a 302 which signals a temporary redirect.

From Technical SEO: why it matters and seven tweaks to give you an edge by John Stevens #424

A permanent redirect that passes link juice from an old URL to a new one, used during site migrations to preserve page authority.

From 5 More Super- Common SEO Mistakes Content Marketers Make by Stephan Spencer #328

A permanent redirect that points an old URL to a new one, preserving link equity and preventing broken links after a site redesign.

From #SEOpodcast 273 - Common SEO Mistakes You Can Avoid

A permanent redirect from one URL to another that passes link juice to the destination, preferred by search engines over a 404 error or other redirect types.

From 7 Content Types That Gain the Most Engagement & Links by Julie McCoy #373

A permanent redirect that tells search engines and users a page has moved to a new URL, preserving link equity when a site migrates.

From Myths vs Facts - Internet Marketing - #SEOpodcast 232

A permanent redirect that passes link juice from an old URL to a new one. The hosts say failing to set up 301 redirects when moving a blog caused a listener to lose significant rankings.

From Learning and Implementing Internet Marketing Basics - Internet Marketing Podcast Number 205

A permanent redirect that maps an old URL to a new one, preserving link equity and ensuring visitors and search engines reach the correct page after a site redesign.

From Website Launch Tips - #seopodcast 215

A permanent redirect telling search engines to stop looking at one URL and treat another as the canonical location.

From Web Development Tips - #SEOpodcast 265

A redirect used to preserve link juice from old pages and pass it to new pages, keeping SEO value while pointing users and crawlers to the current URL.

From Keeping Track of Site Metrics - Internet Marketing Podcast Number 195

A permanent redirect that passes the existing SEO value, called Google Juice in this episode, from an old URL to a new one so rankings are not lost when a page name changes.

A permanent redirect used to consolidate multiple URL variants of the same page so inbound link equity is not split across four different addresses.

From #SEOpodcast Year End Review Part 4-6 - #SEOPodcast 261

A server directive telling search engines a page has permanently moved, consolidating link juice to the final destination URL instead of splitting it across duplicate addresses.

From How to increase your search engine traffic by 25% - Unknown Secrets of SEO E-Webstyle Number 49

A permanent redirect used to point one URL to another, preserving link authority so that links pointing to old domain versions transfer their value to the primary domain.

A permanent redirect used to send visitors and search engines from one URL to another, recommended to be used only once rather than in multiple hops.

From SEO back to the Basics - Internet Marketing Podcast - Number 85

From What is a good redirect vs. a bad redirect, change your webpage or website without losing Google juice - Unknown Secrets of SEO E-Webstyle Number 26

A server instruction that permanently points one URL to another, consolidating link value so Google juice is not split across duplicate pages.

From Choose a CMS like Wordpress for SEO - SEO Podcast - Number 81

A redirect that forwards visitors and search engines from an old URL to a new one, carrying over link value and link juice, especially important when migrating a site to a new platform like WordPress.

From SEO Ideas for the Fall Season - #SEOpodcast 239

A server-side redirect that permanently points one URL to another, consolidating link authority and resolving duplicate content issues such as www versus non-www versions of a site.
